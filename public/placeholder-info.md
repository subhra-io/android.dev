# Public Assets

Add the following files to the `public` directory to complete your portfolio:

## Required Images

### Profile & Projects
- `profile.jpg` - Your professional headshot (400x400px recommended)
- `project-ecommerce.jpg` - E-commerce project screenshot
- `project-taskmanager.jpg` - Task manager project screenshot
- `placeholder-album.png` - Default album artwork for Spotify

### Resume
- `resume.pdf` - Your current resume/CV

## Optional Assets
- `favicon.ico` - Website favicon
- `logo.png` - Your personal logo
- Any additional project screenshots

## Image Guidelines
- **Format**: JPG for photos, PNG for graphics with transparency
- **Size**: Optimize for web (under 1MB each)
- **Quality**: High resolution but compressed for fast loading
- **Naming**: Use descriptive, lowercase names with hyphens

## Quick Setup
1. Add your profile photo as `profile.jpg`
2. Add project screenshots with the names referenced in the code
3. Add your resume as `resume.pdf`
4. Update image paths in components if you use different names
